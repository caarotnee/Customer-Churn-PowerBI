\# 📊 Customer Churn Analysis (Power BI)



\## 📌 Overview



This project analyzes customer churn behavior in a telecom company using Power BI.

The goal is to identify key factors influencing churn and provide actionable insights.

\---



\## 🔍 Key Insights



\* Customers with \*\*short tenure (<12 months)\*\* have the highest churn rate

\* \*\*Month-to-month contracts\*\* lead to significantly higher churn

\* \*\*Fiber optic users\*\* show the highest churn among internet services

\* Customers without \*\*OnlineSecurity\*\* are more likely to leave

\* Higher \*\*monthly charges\*\* correlate with higher churn

\* Customers using \*\*more services\*\* are less likely to churn



\---



\## 💡 Recommendations



\* Improve onboarding for new customers

\* Encourage long-term contracts with incentives

\* Improve Fiber optic service quality or pricing

\* Promote OnlineSecurity bundles

\* Offer service bundles to increase retention

\* Encourage automatic payment methods
